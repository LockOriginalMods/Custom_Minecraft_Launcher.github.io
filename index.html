Minecraft Launcher: Обзор кода

В этой статье мы рассмотрим код для запуска Minecraft с использованием Java. Мы обсудим основные компоненты программы, как она работает и как можно использовать этот код для создания собственного лаунчера Minecraft.

Код представляет собой простой лаунчер для Minecraft, написанный на Java с использованием библиотеки Swing для создания графического интерфейса пользователя (GUI). Лаунчер позволяет пользователю выбирать версию Minecraft для установки и запуска. Давайте разберем все классы, методы и переменные.

Импорты

Код начинается с импорта необходимых библиотек и пакетов. Эти библиотеки включают библиотеки Swing для создания графического интерфейса, библиотеки для работы с JSON и обработки сетевых соединений.

Основной класс MinecraftLauncher наследуется от JFrame (основного класса для создания окна GUI в Swing).

Вспомогательные методы

Метод ensureJavaInstalled(): Загрузка и установка Java

Если Java не установлена, программа автоматически загружает и устанавливает её в нужное место.

private static void ensureJavaInstalled() throws IOException {
    File javaDir = new File(JAVA_PATH);
    if (!javaDir.exists()) {
        javaDir.mkdirs();
        System.out.println("Скачиваем и устанавливаем нужную версию Java...");

        File javaZip = new File(JAVA_PATH + "\\java.zip");
        downloadFileWithProgress(JAVA_DOWNLOAD_URL, javaZip);

        unzip(javaZip.getAbsolutePath(), JAVA_PATH);
        javaZip.delete(); // Удаляем zip-файл после распаковки

        System.out.println("Java успешно установлена.");
    }
}

Метод ensureJavaInstalled(): Проверяет, установлена ли Java, и, если нет, скачивает и устанавливает ее.

  • Создает директорию для Java, если она не существует.
  • Скачивает ZIP-архив с Java и распаковывает его.

Метод fetchMinecraftVersions()Этот метод загружает список доступных версий Minecraft с удалённого сервера и возвращает его в виде списка объектов MinecraftVersion.

  • Возвращаемый тип: List
  • Исключение: IOException — возникает при проблемах с сетью или чтением данных.

Важные переменные:

  • versions — список версий Minecraft.
  • input, reader, jsonReader — потоки для работы с входными данными.
  • jsonObject, versionsArray — объекты для хранения и работы с JSON-данными.
private static List fetchMinecraftVersions() throws IOException {
                List versions = new ArrayList<>();
                try (InputStream input = new URL(BASE_URL).openStream();
                     InputStreamReader reader = new InputStreamReader(input, StandardCharsets.UTF_8);
                     JsonReader jsonReader = new JsonReader(reader)) {
        
                    Gson gson = new Gson();
                    JsonObject jsonObject = gson.fromJson(jsonReader, JsonObject.class);
                    JsonArray versionsArray = jsonObject.getAsJsonArray("versions");
        
                    for (int i = 0; i < versionsArray.size(); i++) {
                        JsonObject versionObject = versionsArray.get(i).getAsJsonObject();
                        String id = versionObject.get("id").getAsString();
                        String url = versionObject.get("url").getAsString();
                        versions.add(new MinecraftVersion(id, url));
                    }
                }
        
                return versions;
            }

Основные действия:

  • Подключается к удалённому серверу через URL.
  • Читает JSON-файл с данными о версиях и парсит его с помощью библиотеки Gson.
  • Для каждой версии Minecraft создаёт объект MinecraftVersion и добавляет его в список.

Метод downloadAndInstallVersion(MinecraftVersion version)Этот метод скачивает и устанавливает конкретную версию Minecraft.

    Параметры

    • version — объект класса MinecraftVersion, представляющий конкретную версию игры.
    • Исключение: IOException — при проблемах с загрузкой или записью файлов.
    private static void downloadAndInstallVersion(MinecraftVersion version) throws IOException {
            String versionPath = MINECRAFT_PATH + "\\versions\\" + version.id;
            File versionDir = new File(versionPath);
            if (!versionDir.exists()) {
                versionDir.mkdirs();
            }
    
            String versionJsonPath = versionPath + "\\" + version.id + ".json";
            File versionJsonFile = new File(versionJsonPath);
            if (!versionJsonFile.exists()) {
                downloadFileWithProgress(version.url, versionJsonFile);
            }
    
            JsonObject versionJson;
            try (Reader reader = new FileReader(versionJsonFile)) {
                Gson gson = new Gson();
                versionJson = gson.fromJson(reader, JsonObject.class);
            }
    
            JsonObject downloads = versionJson.getAsJsonObject("downloads");
            JsonObject client = downloads.getAsJsonObject("client");
            if (client != null) {
                String clientUrl = client.get("url").getAsString();
                String clientPath = versionPath + "\\" + version.id + ".jar";
                File clientJarFile = new File(clientPath);
                if (!clientJarFile.exists()) {
                    downloadFileWithProgress(clientUrl, clientJarFile);
                }
            }
    
            JsonArray libraries = versionJson.getAsJsonArray("libraries");
            String librariesPath = versionPath + "\\libraries"; // Папка libraries теперь уникальна для каждой версии
            for (int i = 0; i < libraries.size(); i++) {
                JsonObject library = libraries.get(i).getAsJsonObject();
                JsonObject downloadsObject = library.getAsJsonObject("downloads");
                if (downloadsObject != null) {
                    if (downloadsObject.has("artifact")) {
                        JsonObject artifact = downloadsObject.getAsJsonObject("artifact");
                        String libraryUrl = artifact.get("url").getAsString();
                        String libraryFilePath = librariesPath + "\\" + artifact.get("path").getAsString();
                        File libraryFile = new File(libraryFilePath);
                        if (!libraryFile.exists()) {
                            downloadFileWithProgress(libraryUrl, libraryFile);
                        }
                    }
    
                    if (downloadsObject.has("classifiers")) {
                        JsonObject classifiers = downloadsObject.getAsJsonObject("classifiers");
                        if (classifiers.has("natives-windows")) {
                            String nativesClassifier = classifiers.get("natives-windows").getAsJsonObject().get("url").getAsString();
                            String nativeFileName = nativesClassifier.substring(nativesClassifier.lastIndexOf('/') + 1);
                            String nativesPath = versionPath + "\\natives\\" + nativeFileName;
                            File nativesFile = new File(nativesPath);
                            if (!nativesFile.exists()) {
                                downloadFileWithProgress(nativesClassifier, nativesFile);
                                unzip(nativesFile.getAbsolutePath(), versionPath + "\\natives");
                            }
                        }
                    }
                }
            }
    
            System.out.println("Версия " + version.id + " успешно установлена.");
        }

    Основные действия:

    • Создаёт директории для файлов версии.
    • Проверяет наличие необходимых файлов и загружает их при отсутствии.
    • Загружает библиотеки и дополнительные файлы, если они требуются для данной версии.
    • Распаковывает архивы с native-библиотеками для Windows.

    Метод downloadFileWithProgressЗагружает файл с указанного URL и показывает прогресс загрузки в консоли.

      Параметры

      • fileURL — URL для загрузки файла.
      • destination — путь к файлу, куда будет сохранён загруженный файл.
      • Исключение: IOException — при проблемах с загрузкой или записью файла.
      private static void downloadFileWithProgress(String fileURL, File destination) throws IOException {
            // Проверяем и создаем необходимые директории
            File parentDir = destination.getParentFile();
            if (parentDir != null && !parentDir.exists()) {
                parentDir.mkdirs(); // Создаем все необходимые директории
            }
    
            URL url = new URL(fileURL);
            try (InputStream inputStream = url.openStream();
                 FileOutputStream outputStream = new FileOutputStream(destination)) {
    
                byte[] buffer = new byte[4096];
                long totalBytesRead = 0;
                int bytesRead;
                long fileSize = url.openConnection().getContentLengthLong();
                int lastPercent = 0;
    
                while ((bytesRead = inputStream.read(buffer)) != -1) {
                    outputStream.write(buffer, 0, bytesRead);
                    totalBytesRead += bytesRead;
                    int percentCompleted = (int) ((totalBytesRead * 100) / fileSize);
                    if (percentCompleted != lastPercent) {
                        System.out.print("\rСкачивание: " + percentCompleted + "%");
                        lastPercent = percentCompleted;
                    }
                }
                System.out.println(); // Переход на новую строку после завершения загрузки
            }
        }

      Основные действия:

      • Создаёт директории, если они отсутствуют.
      • Скачивает файл по частям, показывая прогресс в процентах.

      Метод unzipРаспаковывает архив (ZIP) в указанную директорию, отображая прогресс распаковки.

      Параметры

      • zipFilePath — путь к ZIP-файлу.
      • destDir — директория, куда будут распакованы файлы.
      • Исключение: IOException — при проблемах с загрузкой или записью файла.
      private static void unzip(String zipFilePath, String destDir) throws IOException {
            File dir = new File(destDir);
            if (!dir.exists()) dir.mkdirs();
    
            long totalFiles = 0;
            long filesProcessed = 0;
    
            try (ZipInputStream zipIn = new ZipInputStream(new FileInputStream(zipFilePath))) {
                ZipEntry entry = zipIn.getNextEntry();
                while (entry != null) {
                    if (!entry.isDirectory()) {
                        totalFiles++;
                    }
                    zipIn.closeEntry();
                    entry = zipIn.getNextEntry();
                }
            }
    
            try (ZipInputStream zipIn = new ZipInputStream(new FileInputStream(zipFilePath))) {
                ZipEntry entry = zipIn.getNextEntry();
                while (entry != null) {
                    String filePath = destDir + File.separator + entry.getName();
                    if (!entry.isDirectory()) {
                        extractFile(zipIn, filePath);
                        filesProcessed++;
                        int percentCompleted = (int) ((filesProcessed * 100) / totalFiles);
                        System.out.print("\rРаспаковка: " + percentCompleted + "%");
                    } else {
                        File dirEntry = new File(filePath);
                        dirEntry.mkdirs();
                    }
                    zipIn.closeEntry();
                    entry = zipIn.getNextEntry();
                }
                System.out.println(); // Переход на новую строку после завершения распаковки
            }
        }

      Основные действия:

      • Открывает ZIP-архив и считает количество файлов.
      • Распаковывает файлы, отображая прогресс в консоли.

      Метод extractFileИзвлекает отдельный файл из ZIP-архива и сохраняет его на диск.

      Параметры

      • zipIn — поток для чтения содержимого ZIP-архива.
      • filePath — путь для сохранения извлечённого файла.
      • Исключение: IOException — при проблемах с загрузкой или записью файла.
      private static void extractFile(ZipInputStream zipIn, String filePath) throws IOException {
            try (BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(filePath))) {
                byte[] bytesIn = new byte[4096];
                int read;
                while ((read = zipIn.read(bytesIn)) != -1) {
                    bos.write(bytesIn, 0, read);
                }
            }
        }

      Метод launchMinecraftЭтот метод запускает Minecraft с использованием Java, подставляя необходимые параметры для запуска.

      Параметры

      • version — объект MinecraftVersion, представляющий конкретную версию игры.
      • Исключение: IOException — при проблемах с загрузкой или записью файла.
      private static void launchMinecraft(MinecraftVersion version) throws IOException {
            String versionPath = MINECRAFT_PATH + "\\versions\\" + version.id;
            String javaExecutable = JAVA_PATH + "\\jdk-21+35\\bin\\java.exe"; // Путь к Java
    
            // Проверка существования файла java.exe
            File javaFile = new File(javaExecutable);
            if (!javaFile.exists() || !javaFile.isFile()) {
                System.out.println("Java executable not found at: " + javaExecutable);
                return;
            }
    
            // Сборка classpath: добавляем все .jar из папки libraries и версии
            StringBuilder classpath = new StringBuilder();
            File librariesDir = new File(versionPath + "\\libraries");
            if (librariesDir.exists() && librariesDir.isDirectory()) {
                // Рекурсивно добавляем все jar-файлы из библиотек
                addJarFilesToClasspath(librariesDir, classpath);
            }
    
            // Добавляем jar файл самой версии Minecraft
            classpath.append(File.pathSeparator).append(versionPath).append("\\").append(version.id).append(".jar");
    
            System.out.println("Запуск Minecraft...");
    
            List command = new ArrayList<>();
            command.add(javaExecutable);
            command.add("-cp");
            command.add(classpath.toString());
            command.add("-Djava.library.path=" + versionPath + "\\natives");
            command.add("net.minecraft.client.main.Main");
            command.add("--username");
            command.add("MailoDesx");
            command.add("--version");
            command.add(version.id);
            command.add("--gameDir");
            command.add(MINECRAFT_PATH);
            command.add("--assetsDir");
            command.add(MINECRAFT_PATH + "\\assets");
            command.add("--assetIndex");
            command.add(version.id);
            command.add("--uuid");
            command.add("1234567890abcdef1234567890abcdef");
            command.add("--accessToken");
            command.add("access_token");
            command.add("--userProperties");
            command.add("{}");
    
            ProcessBuilder processBuilder = new ProcessBuilder(command);
            processBuilder.redirectErrorStream(true);
    
            // Запуск процесса
            Process process = processBuilder.start();
            System.out.println("Minecraft запущен!");
    
            // Чтение вывода процесса
            new Thread(() -> {
                try (var reader = new BufferedReader(new InputStreamReader(process.getInputStream()))) {
                    String line;
                    while ((line = reader.readLine()) != null) {
                        System.out.println(line);
                    }
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }).start();
        }

      Основные действия:

      • Проверяет наличие исполняемого файла java.exe.
      • Собирает classpath, включая все библиотеки и jar-файл Minecraft.
      • Формирует команду для запуска Minecraft и выполняет её с помощью ProcessBuilder.
      • Запускает поток для вывода сообщений из процесса.

      Метод addJarFilesToClasspathЭтот метод рекурсивно добавляет все JAR-файлы из указанной директории в classpath.

      Параметры

      • dir — директория, в которой находятся JAR-файлы.
      • classpath — строка, представляющая classpath, в которую добавляются пути к JAR-файлам.
      private static void addJarFilesToClasspath(File dir, StringBuilder classpath) {
            File[] files = dir.listFiles();
            if (files != null) {
                for (File file : files) {
                    if (file.isDirectory()) {
                        addJarFilesToClasspath(file, classpath);

                private static List fetchMinecraftVersions() throws IOException {
                    List versions = new ArrayList<>();
                    try (InputStream input = new URL(BASE_URL).openStream();
                         InputStreamReader reader = new InputStreamReader(input, StandardCharsets.UTF_8);
                         JsonReader jsonReader = new JsonReader(reader)) {

                        JsonArray versionsArray = jsonObject.getAsJsonArray("versions");

                            String id = versionObject.get("id").getAsString();
                            String url = versionObject.get("url").getAsString();

                private static void downloadAndInstallVersion(MinecraftVersion version) throws IOException {
                    String versionPath = MINECRAFT_PATH + "\\versions\\" + version.id;
                    File versionDir = new File(versionPath);
                    if (!versionDir.exists()) {
                        versionDir.mkdirs();
                    }
            
                    String versionJsonPath = versionPath + "\\" + version.id + ".json";
                    File versionJsonFile = new File(versionJsonPath);
                    if (!versionJsonFile.exists()) {
                        downloadFileWithProgress(version.url, versionJsonFile);
                    }
            
                    JsonObject versionJson;
                    try (Reader reader = new FileReader(versionJsonFile)) {
                        Gson gson = new Gson();
                        versionJson = gson.fromJson(reader, JsonObject.class);
                    }
            
                    JsonObject downloads = versionJson.getAsJsonObject("downloads");
                    JsonObject client = downloads.getAsJsonObject("client");
                    if (client != null) {
                        String clientUrl = client.get("url").getAsString();
                        String clientPath = versionPath + "\\" + version.id + ".jar";
                        File clientJarFile = new File(clientPath);
                        if (!clientJarFile.exists()) {
                            downloadFileWithProgress(clientUrl, clientJarFile);
                        }
                    }
            
                    JsonArray libraries = versionJson.getAsJsonArray("libraries");
                    String librariesPath = versionPath + "\\libraries"; // Папка libraries теперь уникальна для каждой версии
                    for (int i = 0; i < libraries.size(); i++) {
                        JsonObject library = libraries.get(i).getAsJsonObject();
                        JsonObject downloadsObject = library.getAsJsonObject("downloads");
                        if (downloadsObject != null) {
                            if (downloadsObject.has("artifact")) {
                                JsonObject artifact = downloadsObject.getAsJsonObject("artifact");
                                String libraryUrl = artifact.get("url").getAsString();
                                String libraryFilePath = librariesPath + "\\" + artifact.get("path").getAsString();
                                File libraryFile = new File(libraryFilePath);
                                if (!libraryFile.exists()) {
                                    downloadFileWithProgress(libraryUrl, libraryFile);
                                }
                            }
            
                            if (downloadsObject.has("classifiers")) {
                                JsonObject classifiers = downloadsObject.getAsJsonObject("classifiers");
                                if (classifiers.has("natives-windows")) {
                                    String nativesClassifier = classifiers.get("natives-windows").getAsJsonObject().get("url").getAsString();
                                    String nativeFileName = nativesClassifier.substring(nativesClassifier.lastIndexOf('/') + 1);
                                    String nativesPath = versionPath + "\\natives\\" + nativeFileName;
                                    File nativesFile = new File(nativesPath);
                                    if (!nativesFile.exists()) {
                                        downloadFileWithProgress(nativesClassifier, nativesFile);
                                        unzip(nativesFile.getAbsolutePath(), versionPath + "\\natives");
                                    }
                                }
                            }
                        }
                    }
            
                    System.out.println("Версия " + version.id + " успешно установлена.");
                }
            
            
                private static void downloadFileWithProgress(String fileURL, File destination) throws IOException {
                    // Проверяем и создаем необходимые директории
                    File parentDir = destination.getParentFile();
                    if (parentDir != null && !parentDir.exists()) {
                        parentDir.mkdirs(); // Создаем все необходимые директории
                    }
            
                    URL url = new URL(fileURL);
                    try (InputStream inputStream = url.openStream();
                         FileOutputStream outputStream = new FileOutputStream(destination)) {
            
                        byte[] buffer = new byte[4096];
                        long totalBytesRead = 0;
                        int bytesRead;
                        long fileSize = url.openConnection().getContentLengthLong();
                        int lastPercent = 0;
            
                        while ((bytesRead = inputStream.read(buffer)) != -1) {
                            outputStream.write(buffer, 0, bytesRead);
                            totalBytesRead += bytesRead;
                            int percentCompleted = (int) ((totalBytesRead * 100) / fileSize);
                            if (percentCompleted != lastPercent) {
                                System.out.print("\rСкачивание: " + percentCompleted + "%");
                                lastPercent = percentCompleted;
                            }
                        }
                        System.out.println(); // Переход на новую строку после завершения загрузки
                    }
                }
            
                private static void unzip(String zipFilePath, String destDir) throws IOException {
                    File dir = new File(destDir);
                    if (!dir.exists()) dir.mkdirs();
            
                    long totalFiles = 0;
                    long filesProcessed = 0;
            
                    try (ZipInputStream zipIn = new ZipInputStream(new FileInputStream(zipFilePath))) {
                        ZipEntry entry = zipIn.getNextEntry();
                        while (entry != null) {
                            if (!entry.isDirectory()) {
                                totalFiles++;
                            }
                            zipIn.closeEntry();
                            entry = zipIn.getNextEntry();
                        }
                    }
            
                    try (ZipInputStream zipIn = new ZipInputStream(new FileInputStream(zipFilePath))) {
                        ZipEntry entry = zipIn.getNextEntry();
                        while (entry != null) {
                            String filePath = destDir + File.separator + entry.getName();
                            if (!entry.isDirectory()) {
                                extractFile(zipIn, filePath);
                                filesProcessed++;
                                int percentCompleted = (int) ((filesProcessed * 100) / totalFiles);
                                System.out.print("\rРаспаковка: " + percentCompleted + "%");
                            } else {
                                File dirEntry = new File(filePath);
                                dirEntry.mkdirs();
                            }
                            zipIn.closeEntry();
                            entry = zipIn.getNextEntry();
                        }
                        System.out.println(); // Переход на новую строку после завершения распаковки
                    }
                }
            
                private static void extractFile(ZipInputStream zipIn, String filePath) throws IOException {
                    try (BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(filePath))) {
                        byte[] bytesIn = new byte[4096];
                        int read;
                        while ((read = zipIn.read(bytesIn)) != -1) {
                            bos.write(bytesIn, 0, read);
                        }
                    }
                }
            
                private static void launchMinecraft(MinecraftVersion version) throws IOException {
                    String versionPath = MINECRAFT_PATH + "\\versions\\" + version.id;
                    String javaExecutable = JAVA_PATH + "\\jdk-21+35\\bin\\java.exe"; // Путь к Java
            
                    // Проверка существования файла java.exe
                    File javaFile = new File(javaExecutable);
                    if (!javaFile.exists() || !javaFile.isFile()) {
                        System.out.println("Java executable not found at: " + javaExecutable);
                        return;
                    }
            
                    // Сборка classpath: добавляем все .jar из папки libraries и версии
                    StringBuilder classpath = new StringBuilder();
                    File librariesDir = new File(versionPath + "\\libraries");
                    if (librariesDir.exists() && librariesDir.isDirectory()) {
                        // Рекурсивно добавляем все jar-файлы из библиотек
                        addJarFilesToClasspath(librariesDir, classpath);
                    }
            
                    // Добавляем jar файл самой версии Minecraft
                    classpath.append(File.pathSeparator).append(versionPath).append("\\").append(version.id).append(".jar");
            
                    System.out.println("Запуск Minecraft...");
            
                    List command = new ArrayList<>();
                    command.add(javaExecutable);
                    command.add("-cp");
                    command.add(classpath.toString());
                    command.add("-Djava.library.path=" + versionPath + "\\natives");
                    command.add("net.minecraft.client.main.Main");
                    command.add("--username");
                    command.add("MailoDesx");
                    command.add("--version");
                    command.add(version.id);
                    command.add("--gameDir");
                    command.add(MINECRAFT_PATH);
                    command.add("--assetsDir");
                    command.add(MINECRAFT_PATH + "\\assets");
                    command.add("--assetIndex");
                    command.add(version.id);
                    command.add("--uuid");
                    command.add("1234567890abcdef1234567890abcdef");
                    command.add("--accessToken");
                    command.add("access_token");
                    command.add("--userProperties");
                    command.add("{}");
            
                    ProcessBuilder processBuilder = new ProcessBuilder(command);
                    processBuilder.redirectErrorStream(true);
            
                    // Запуск процесса
                    Process process = processBuilder.start();
                    System.out.println("Minecraft запущен!");
            
                    // Чтение вывода процесса
                    new Thread(() -> {
                        try (var reader = new BufferedReader(new InputStreamReader(process.getInputStream()))) {
                            String line;
                            while ((line = reader.readLine()) != null) {
                                System.out.println(line);
                            }
                        } catch (IOException e) {
                            e.printStackTrace();
                        }
                    }).start();
                }
            
            
                private static void addJarFilesToClasspath(File dir, StringBuilder classpath) {
                    File[] files = dir.listFiles();
                    if (files != null) {
                        for (File file : files) {
                            if (file.isDirectory()) {
                                addJarFilesToClasspath(file, classpath);
                            } else if (file.getName().endsWith(".jar")) {
                                classpath.append(file.getAbsolutePath()).append(File.pathSeparator);
                            }
                        }
                    }
                }
